This repository contains the tasks done for the [Quantitative Research Virtual Internship at JPMorgan Chase & Co.](https://www.theforage.com/simulations/jpmorgan/quantitative-research-11oc)

## Overview

This project leverages quantitative research methods, machine learning, and data analysis to address challenges in the financial and energy sectors. The analysis focuses on developing pricing models for natural gas contracts, estimating loan default probabilities, and predicting losses for banks. Key methods such as K-means clustering, machine learning techniques, and time series decomposition were employed to provide actionable insights

## Project Highlights

- Developed a prototype pricing model for natural gas contracts, based on the estimated prices at which the commodity can be purchased/sold on specific dates, the rate at which gas can be injected/withdrawn from storage, and its costs.

- Estimated the probability of loan defaults and the expected loss for a bank by analyzing a book of loans and using machine learning techniques.

- Used quantization to categorize FICO scores into a smaller set of rating buckets, making the data easier to work with and ensuring the model can be applied to future datasets.
